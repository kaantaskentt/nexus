# frontend/

Next.js 14 (app router) UI. Design system V2 (A16 "10k designer" bar) lives in `src/app/globals.css` + `tailwind.config.ts` (A15.1 — no ad-hoc colors): a warm surface ladder, a 4-step layered elevation scale (`--elev-1..4`), glass for floating chrome (`.glass` — top bar, drawer, sticky rails), a type scale, a density dial (`data-density`), and a motion vocabulary. Shared framer-motion variants (rise / stagger / drawer-spring) are in `src/lib/variants.ts` so motion is consistent, not per-component; `prefers-reduced-motion` is honored globally. Trust rules are enforced in `src/components/` (confidence badges, paraphrased employee evidence, facts-only why-lines). Entry: `npm run dev`; screens under `src/app/w/[slug]/` (admin) and `src/app/i/[token]/` (respondent); data is live via `src/lib/live.ts` against the API at `:8000` — conversation paths never mock.
